# How to Export on Android Since you don't have a physical keyboard, follow these steps to get your code: 1. **Select the Screen**: Tap once on the design you want to export. It will be highlighted with a blue border. 2. **Open the Code Viewer**: Look at the top toolbar and tap the button labeled **" View Code"**. 3. **Copy the Content**: A window will slide up containing all the HTML and CSS. You can select all the text and copy it directly to your clipboard. 4. **Repeat for Manifests**: Do the same for the **DESIGN.md** and **Master Asset Manifest** documents so your no-code agent has all the instructions. **Tip:** If you need to select multiple screens, look for a "Select Multiple" toggle or long-press the screens in the order you want them.
